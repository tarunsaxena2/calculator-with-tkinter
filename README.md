# calculator-with-tkinter
This is a basic calculator made using Python and Tkinter.

I created this project to understand:

- How to create a Tkinter window
- How buttons work
- How to use functions with buttons
- How to display numbers in an input box
- How to perform basic calculations
- How to handle incorrect input

## Features

The calculator can perform:

- Addition
- Subtraction
- Multiplication
- Division
- Decimal calculations
- Clear calculation
- Delete last character

## How to Run

1. Install Python on your computer.

2. Open the calculator project folder.

3. Open Command Prompt or PowerShell inside the folder.

4. Run this command:

```bash
python calculater.py
